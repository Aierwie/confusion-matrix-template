# Styled Confusion Matrix

A visually enhanced confusion matrix plotting tool for classification tasks.

---

## 🚀 Usage

```python
# 1. 修改你的混淆矩阵（或 y_true / y_pred）
cmat = np.array([
    [5,1,0],
    [0,6,1],
    [0,0,7]
])

# 2. 设置类别名称
class_names = ["Class A", "Class B", "Class C"]

# 3. 运行脚本
python confusion_matrix_plot.py
```

---

## ✨ Features

* Rounded grid with shadow effect
* Dual color scheme:

  * Red → correct predictions
  * Blue → errors
* Clean and publication-style layout
* Automatic font selection
